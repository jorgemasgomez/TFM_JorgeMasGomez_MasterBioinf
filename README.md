## Description

This repository contains the bioinformatics and statistical analysis workflows developed for the Master's Thesis:

**Genome-wide association study for morphological and nutritional traits in almond using high-throughput genotyping and phenotyping**

The project focuses on the integration of high-throughput genotyping and phenotyping data to investigate the genetic basis of morphological, physical and nutritional traits in almond (*Prunus dulcis*). The workflow includes phenotypic data processing, estimation of BLUEs and BLUPs, variance component analysis, broad-sense heritability estimation, genome-wide association studies, candidate gene annotation, GWAS visualization and allelic effect analysis.

The scripts are designed as reusable templates for analyzing almond trait datasets generated from image-based phenotyping, nutritional prediction models and genomic marker data.

## Repository structure

### `TFM_template_histograms_heritability_BLUEs_plots.R`

Template script for phenotypic data analysis. It includes:

* Data loading and preprocessing.
* Descriptive histograms by environment or year.
* Outlier detection using the MAD modified Z-score method.
* Estimation of BLUEs and BLUPs using ASReml.
* Variance component estimation.
* Broad-sense heritability calculation.
* Publication-quality plots for trait distributions, BLUEs and variance components.

### `clean_merge_BLUEs_BLUPs_heritability_plots.R`

Script for merging and summarizing outputs from multiple phenotypic analyses. It includes:

* Merging variance component tables.
* Merging BLUE and BLUP result files.
* Formatting BLUEs into a wide matrix for downstream analyses.
* Generating stacked variance component plots.
* Producing BLUP-based correlation heatmaps.
* Performing PCA using BLUP values.

### `01_run_GWAS_mrMLM_and_merge_results.R`

Script for running genome-wide association studies using the `mrMLM` package. It includes:

* GWAS execution using multiple multi-locus methods.
* Support for phenotype, genotype and population structure files.
* Automatic use of kinship estimation when no kinship matrix is provided.
* Merging of all final GWAS result files into a single output table.

Main input files:

* `myG.txt`: genotype file.
* `myY.txt`: phenotype file.
* `myCV_K9.txt`: covariate or population structure file.

Main outputs:

* GWAS result folders generated by `mrMLM`.
* Merged GWAS result table.

### `02_annotate_GWAS_candidate_genes.R`

Script for candidate gene annotation around significant GWAS markers. It includes:

* Creation of genomic windows around significant SNPs.
* Identification of nearby genes using a GFF3 annotation file.
* Extraction of gene-level functional annotations.
* Annotation using Ensembl Plants BioMart.
* Retrieval of GO, KEGG, InterPro, UniProt and WikiGene information.
* Identification of Arabidopsis orthologs.
* Integration of Arabidopsis functional annotations.
* Generation of collapsed and final candidate gene annotation tables.

Main input files:

* Merged GWAS results.
* Almond genome annotation file in GFF3 format.

Main outputs:

* Candidate genes around GWAS markers.
* Functional annotation tables.
* Final consolidated candidate gene annotation file.

### `03_create_GWAS_circos_plot.R`

Script for creating a multivariate GWAS circos plot. It includes:

* Visualization of GWAS associations across the eight almond chromosomes.
* One Manhattan-style circular track per trait group.
* SNP density track.
* Hotspot detection by genomic windows.
* Hotspot links between genomic regions.
* Trait-specific color coding.
* R²-based point size scaling.
* Compact legend generation.

Main output:

* High-resolution circos plot for GWAS summary visualization.

### `04_plot_allelic_effect_boxplots.R`

Script for visualizing allelic effects of selected GWAS markers. It includes:

* Extraction of marker genotypes from HapMap files.
* Integration of genotype and phenotype or BLUE data.
* Generation of violin plots, boxplots and jittered individual points.
* Sample size labels per genotype class.
* Automatic export of one plot per marker-trait combination.

Main input files:

* `myG.txt`: genotype file.
* `myY.txt`: phenotype or BLUE table.
* `marcadores_selected.txt`: selected marker-trait combinations.

Main outputs:

* Allelic effect plots for selected markers and traits.

## General workflow

The complete analysis workflow can be summarized as follows:

1. **Phenotypic data processing**
   Raw phenotypic data are inspected, cleaned and summarized across environments or years.

2. **BLUE and BLUP estimation**
   Mixed models are used to estimate adjusted trait values and genotype effects.

3. **Heritability estimation**
   Variance components are estimated to calculate broad-sense heritability for each trait.

4. **GWAS analysis**
   Multi-locus GWAS is performed using genotype, phenotype and population structure data.

5. **Candidate gene annotation**
   Significant GWAS markers are linked to nearby genes and functionally annotated.

6. **GWAS visualization**
   Significant associations are summarized using circos plots and hotspot detection.

7. **Allelic effect analysis**
   Selected marker-trait associations are explored using genotype-specific phenotypic distributions.
