## Description

This repository contains the code, data processing workflows, and analysis scripts developed for the project:

**Genome-wide association study for morphological and nutritional traits in almond using high-throughput genotyping and phenotyping**

The project focuses on the integration of high-throughput genotyping and phenotyping approaches to study the genetic basis of morphological and nutritional traits in almond.
